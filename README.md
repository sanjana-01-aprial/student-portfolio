# AWS CI/CD Pipeline Project

## Project Title
CI/CD Pipeline with AWS CodePipeline, CodeBuild, and CodeDeploy

## Project Description
This project demonstrates an automated CI/CD pipeline for a Student Portfolio website using AWS services. Whenever code is updated in GitHub, AWS automatically builds and deploys the latest version to EC2 instances.

## AWS Services Used
- Amazon EC2
- Auto Scaling Group
- Application Load Balancer (ALB)
- Launch Template
- Amazon Machine Image (AMI)
- AWS CodePipeline
- AWS CodeBuild
- AWS CodeDeploy
- Amazon S3
- IAM
- GitHub

## Project Workflow
GitHub → CodePipeline → CodeBuild → Amazon S3 → CodeDeploy → EC2 → Load Balancer
